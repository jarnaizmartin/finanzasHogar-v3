Architecture

Pourquoi sans cloud ?

Parce que vos données financières sont les plus sensibles qui soient. Votre patrimoine, vos dettes, vos habitudes de dépense. Vous ne devriez pas avoir à faire confiance à un serveur externe pour qu'il ne les voie pas, ne les vende pas, ne les perde pas dans une faille de sécurité.

Local-first signifie que l'application fonctionne entièrement sur votre appareil. Pas de compte à créer, pas de serveur auquel se connecter. Vos données vivent sur votre machine, chiffrées avec AES-GCM 256 + PBKDF2-SHA-256 avec 200 000 itérations. Le même standard que les gestionnaires de mots de passe professionnels.

Le code de chiffrement est public et auditable — c'est l'un de nos arguments marketing, pas une promesse vague.
